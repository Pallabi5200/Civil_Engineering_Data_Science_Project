# Construction Intelligence Platform (CIP)

An end-to-end Data Science and Business Intelligence platform built using real-world civil engineering project data. This enterprise-grade framework transitions unstructured document streams (BOQs, Work Orders, Invoices) into structured relational databases, predictive machine learning pipelines, and actionable executive dashboards.

## 📌 Project Overview
In the construction and civil engineering sectors, critical operational data is frequently siloed within thousands of legacy documents—leading to cost overruns, delayed material tracking, and suboptimal vendor management. 

This project designs and implements a centralized data solution for a mid-sized civil engineering enterprise, focusing on:
* **Data Architecture:** Transforming raw operational documents into a robust, normalized relational database.
* **Analytics Engine:** Utilizing SQL and Python to extract key performance indicators (KPIs) regarding procurement, billing cycles, and operational efficiency.
* **Predictive Modeling:** Developing machine learning models to identify project risks and optimize material tracking.
* **Decision Support:** Building an interactive dashboard and an AI Construction Assistant to provide stakeholders with real-time operational visibility.
